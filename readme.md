# 這是一個小工具 用於降低開機時又要進windows設定中卡開網路連線的小工具
我將cmd連線指令打包成.bat檔 且用工作排程器設定使用時機 讓網路連線不需要手動操作


---
## 使用方法
### 設置.bat檔案
將PPPoE_ez_to_connect.txt檔打開並照著你的 網路名稱 帳號 密碼 路徑依次更改輸入後 把副檔名改成.bat檔 最後測試執行無誤後(將網路斷開後 運行檔案如果連接成功為主) 再到工作排程器設置時機

---

### 工作排程器設定方法:
win+R 輸入 taskschd.msc

- 第一步
建立工作(不是建立基本工作)
打上名稱(必填) 與描述(選填 以免你不知道這是甚麼) 
在安全性選項中勾選以最高權限執行(I) next(跳轉至 觸發程序) 

- 第二步
新增(N)
開始工作(G): 登入時 (可以選填特定使用者) 
進階設定記得勾選已啟用(B) 
按確定 next(跳轉至 動作)

- 第三步
新增(N) 
動作: 啟動程式
設定:
程式或指令碼(P): 瀏覽你的cmd (C:\Windows\System32\cmd.exe) 
新增引述(A): 寫/c + 你的檔案位置(PPPoE_ez_to_connect) 注意是.bat檔 
按確定 next

---

### 如果不想設置工作排成器
- 那就更改 PPPoE_taskschd.xml
(用文字編輯器開啟(VScode等))
更改內容 使用者id 動作名稱 檔案路徑
最後 匯入工作排程器中
